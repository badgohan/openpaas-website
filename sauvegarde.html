Install it, Try it

Try it from Docker

Want to run OpenPaaS with Docker? This is easy with the help of docker-compose! Once you have the sources, just run:
PROVISION=true DOCKER_IP=[YOUR_DOCKER_IP] docker-compose -f ./docker/dockerfiles/platform/docker-compose-images.yml up
Where YOUR_DOCKER_IP is localhost on linux and probably 192.168.99.100 if you use the default docker machine on OS X and Windows.
Launching the platform may take some time (1-2 minutes), grab a coffee and be ready for the next steps!​
​
​

Try it with our Virtual Machine

You can also download a pre-built VirtualBox virtual machine. Download it, import it, add the following line to your hosts file on your local computer ip.address.of.vm openpaas-demo-vm.
​You can then point your favorite browser to http://openpaas-demo-vm
and log in with admin@open-paas.org and password secret.
